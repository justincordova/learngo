## Basic data types and literals

**Integer literal:** A whole number like `-42` or `100`

**Float literal:** A decimal number like `3.14` or `.28` (leading zero optional)

**String literal:** Text wrapped in quotes like `"Hello"`

**Bool constants:** Either `true` or `false`

## Examples

```go
var age int = 42              // integer
var price float64 = 3.14      // float
var name string = "Alice"     // string
var isActive bool = true      // bool
```
